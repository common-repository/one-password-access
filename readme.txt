== Description ==

Only people who have the good password would be able to access your WordPress site.
No users management, only one unique password to access your site content.
I use this plugin to manage the access to a extranet. Any person with the correct password can access the extranet.

== Installation ==

To install Login Manager

1. Upload `login_manager` directory in to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Options -> One Password Access and choose a good password
